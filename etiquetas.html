HTML CSS JAVASCRIPT
< body > : para el contenido color : color de texto Let : Declara una variable con ámbito de bloque
< head > : Para informacion sobre el documento font-size : Tamaño de fuente const : Declara una constante (no se puede reasignar)
< div > : Division dentro del contenido font-family : Tipo de fuente ( Arial, Roboto, etc) var : Antiguo metodo para declarar variables (evitar si es posible)
< a > : Para enlaces font-weight : Grosor del texto const persona : {Nombre: "Ana", Edad: 30}
< strong > : Para poner texto en negrita font-style : Estilo de la fuente null : Valor nulo
< br > : Salto de linea en un texto text-align : Alineacion del texto if, else : Condiciones
< h1 > : Titulos de contenido (tamaño) text-decoration : Subrayado tachado o ninguno continue : Salta a la siguiente interaccion
< img > : Añadir imagenes al documento text-transform : Cambia mayúsculas y minusculas margin : Espacio externo entre el elemento y otros
    : Para listas desordenadas
line-height : Espacio entre lineas de texto switch : Multiples condiciones segun el valor de una variable
< li > : Elementos dentro de la lista letter-spacing : Espacio entre letras submit : Al enviar un formulario
< p > : Para añadir texto o parrafos word-spacing : Espacio entre palabras keydown : Al precionar una tecla
< span > : Para estilos de una partedel texto background-color : Color de fondo alert : Muestra una alerta de emergencia
< title > : Titulo de la pagina background-image : Imagen de fondo .style : Cambia el estilo CSS directamente
< ol > : Listas ordenadas background-size : Tamaño de fondo .innerHTML : Cambia el contenido HTML de un elemento
< link > : Vincula recursos externos a una pagina html como hojas css etc. background-repeat : Repetir o no la imagen de fondo break : Sale de un bucle
< meta > : Define informacion del documento, titulo, referencias etc. opacity : Transparencia del elemento JSON.parse : Convierte una cadena de texto en formato JSON nuevamente a un objeto o arreglo en JavaScript.
< nav > : Define una seccion o grupo de contenido width : Ancho del elemento JSON.stringify : Convierte un objeto o arreglo en una cadena de texto en formato JSON
< html > : Lenguaje marcado que da estructura al contenido height : Alto del elemento setTimeout : Ejecuta una función después de un período de tiempo determinado en milisegundos.
< button > : Crea un botón clicable (enviar, acciones, etc.). display : Define cómo se muestra un elemento (block, inline, flex, etc.). querySelector : Selecciona el primer elemento que coincide con un selector CSS especificado.
< form > : Define un formulario para enviar datos. padding : Espacio interno entre el contenido y el borde. addEventListener : agrega un evento a un elemento HTML para acciones como clics, teclas presionadas, etc.
