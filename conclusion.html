CONCLUSION

Conclusión del Módulo 1.5: Desarrollo de Páginas Web En el módulo 1.5 se adquieren los conocimientos fundamentales para iniciar en el mundo del desarrollo web. A lo largo de este módulo, se aprende a estructurar páginas utilizando HTML (Lenguaje de Marcado de Hipertexto), dando forma y contenido a los sitios web. Además, se introducen conceptos básicos de CSS (Hojas de Estilo en Cascada), permitiendo aplicar estilos visuales como colores, fuentes, márgenes y diseños. También se explora el uso de etiquetas semánticas, enlaces, listas, tablas, formularios y la inserción de imágenes, elementos esenciales para crear páginas web funcionales y atractivas. Este módulo no solo desarrolla habilidades técnicas, sino que también fomenta la creatividad, el pensamiento lógico y la resolución de problemas. En resumen, el módulo 1.5 proporciona las bases necesarias para que el estudiante comience a construir sitios web estáticos de manera estructurada, clara y con principios básicos de diseño, abriendo el camino hacia el desarrollo web profesional.
